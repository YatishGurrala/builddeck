# BuilddeckWorkspace

BuilddeckWorkspace is a lightweight founder/product operating system for products, roadmaps, tasks, docs, and launches.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn/ui styling foundation
- Supabase Auth and Postgres
- Prisma ORM
- Zustand
- React Hook Form + Zod
- TanStack Table
- DnD Kit
- Recharts

## First pass included

- App router scaffold
- Protected dashboard shell
- Sidebar navigation and breadcrumb layout
- Supabase auth helpers
- Prisma schema for the MVP models
- Placeholder pages for dashboard, products, roadmap, tasks, docs, team, and settings

## Next step

Install dependencies, add your Supabase and Postgres env vars in `.env.local`, and wire the first form flows.
